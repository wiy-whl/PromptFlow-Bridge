# PromptFlow Bridge

**The Ultimate LLM Automation & Relay Tool.**  
**让 AI 们自动“传球”的生产力利器。**

PromptFlow Bridge 是一个强大的浏览器扩展，旨在打破不同大模型（LLM）平台之间的围墙。通过高保真的“灵动岛”交互设计，它能让你在 DeepSeek、Kimi、ChatGPT、Gemini 和 Grok 之间实现丝滑的内容接力与自动化流转。

---

## ✨ 核心特性 (Key Features)

- **🏝️ Dynamic Island UI (灵动岛)**: 基于毛玻璃美学的悬浮交互中心，状态感知的 Quick Inject 按钮。
- **🔗 Smart Relay (智能接力)**: 自动抓取当前回答，一键转发至其他模型，支持自动跳转与自动发送。
- **🛡️ Anti-Drift Pipelines (防御性链路)**: 内置锚定机制，确保多级接力后意图不漂移。
- **💎 LaTeX & Markdown Integrity**: 100% 还原数学公式与排版，绝不因传输产生形变。
- **⚙️ Custom Workflows**: 自由组合你的 AI 智囊团，支持自定义提示词模板。

## 🛠️ 安装方法 (Installation)

1.  下载本项目并解压。
2.  打开 Chrome 浏览器，进入 `chrome://extensions/`。
3.  开启右上角的“开发者模式”。
4.  点击“加载已解压的扩展程序”，选择 `dist` 文件夹。
